This is a starter project for the Fastor React assignment with PWA support.

## Features
- Vite + React
- Tailwind CSS
- Routing (react-router-dom)
- Mock API for restaurants
- Canvas image superimpose with draggable logo
- Web Share API
- Basic PWA manifest + service worker (installable)

## Quick start
1. `npm install`
2. `npm run dev`
3. Open the app at the printed localhost URL

Note: Replace `public/fastor-logo.png` with your actual logo.
